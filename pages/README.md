## Deployment

This site is designed for GitHub Pages. Enable Pages in repository settings, set source to main branch.

## Before Going Live

1. Upload PDFs to `assets/pdfs/short-stories/` and `assets/pdfs/poetry/`
2. Configure newsletter form endpoint (replace YOUR_FORM_ID in footer forms)
3. Optionally replace Wix-hosted images with local assets
